# Regras Internas

## UI/UX
- **Modal de Confirmação**: Sempre que uma funcionalidade exigir confirmação ou escolha (ex: Sim/Não), utilize o **modal padrão** do projeto. Evite o uso de `Alert.alert` ou diálogos nativos.

## Manipulação de Dados
- **Ordenação de Listas**: Listas contendo nomes devem exibir os itens em **ordem alfabética**, a menos que outra ordenação seja explicitamente solicitada.

## Idioma
- **Regra de Idioma**: Sempre responda em **português (Brasil)** para todas as solicitações, explicações e comunicações, independentemente do idioma em que a pergunta foi feita.
